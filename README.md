# SEO Pro Analyzer

Công cụ phân tích hiệu quả SEO và tối ưu hóa AI cho website của bạn.

## Tính năng
- Chấm điểm SEO tổng quát dựa trên Google Lighthouse.
- Phân tích tối ưu hóa AI (GEO/AIO).
- So sánh tối đa 3 đối thủ thực tế.
- Thống kê traffic và nguồn traffic trực quan.
- Phân tích từ khóa và độ khó.

## Công nghệ sử dụng
- Next.js 14
- Tailwind CSS
- Framer Motion (Animation)
- Recharts (Biểu đồ)
- Lucide React (Icons)

## Triển khai (Deployment)

### 1. Đẩy code lên GitHub
- Khởi tạo git: `git init`
- Thêm file: `git add .`
- Commit: `git commit -m "Initial commit"`
- Tạo repo trên GitHub và copy link repo.
- Thêm remote: `git remote add origin <link-repo-cua-ban>`
- Push code: `git push -u origin main`

### 2. Kết nối với Vercel
- Truy cập [vercel.com](https://vercel.com) và đăng nhập.
- Nhấn **"Add New..."** -> **"Project"**.
- Chọn repository GitHub vừa tạo.
- Giữ nguyên các thiết lập mặc định (Vercel tự nhận diện Next.js).
- Nhấn **"Deploy"**.
